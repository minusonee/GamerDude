# ROG Challenge 2024 - Game Project

This is a game created for the **ROG Challenge 2024**.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/minusonee/ROG.git
   cd ROG
   ```

2. Open the project in godot.
